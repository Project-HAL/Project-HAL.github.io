class stimulation.ArbitraryStimulation(token: str, sequence_list: List[ApiObject])

Bases: object

A class that holds any stimulation that could be passed to the MaxLab system. Initialized with a list of ApiObject. It remains unclear whether the sample rate is always 20000 samples per second when generating stimuli, regardless of the system being used.

classmethod combine_stimuli(stim1: Self, stim2: Self, token: str, maxtwo=False) → ArbitraryStimulation

Combine two Stimulation objects into a sequence that runs them simultaneously.

Args:

stim1 (Self): The Stimulation object to run through dac source 0. stim2 (Self): The Stimulation object to run through dac source 1. token (str): The unique identifier token for the sequence. initial_delay (float, optional): The delay before the sequence, in ms. Defaults to 5. persistent(bool, optional): Whether or not the sequence exists on the server after its scope is complete. Defaults to True. maxtwo (bool, optional): whether or not the maxtwo is being used. If True, fps is equal to 10000.

Returns:

ArbitraryStimulation: The ArbitraryStimulation object that runs both stim1 and stim2 simultaneously.

Notes:

Not configured to deal with mx.DelayCycles objects.

constructor method, using frequency and duration instead of phase and pulses per train.

Args:

token (str): unique identifier dac_source (int): The DAC channel to send signals to. Must be 0, 1, or 2 to send to a single DAC channel. frequency (float): Stimulation frequency, in Hz. duration (float): How long the stimulus should last (for a single train), in s. amplitude (float, optional): Amplitude of oscillation, in mV. Suggested value between 0 and 1000. Defaults to 100. num_trains (int, optional): Number of pulse trains. Defaults to 1. pulse_train_interval (float, optional): Time between pulse trains, in s. Defaults to 1. stim_units (list, optional): List of stimulation stim_units. Must have a length of less than 32. If None, just sends stimuli to the desired DAC channel. Otherwise, powering on and off stim units is included in the sequence.

IF THIS IS USED, ENSURE THAT ALL STIM_UNITS ARE POWERED OFF TO BEGIN WITH.
